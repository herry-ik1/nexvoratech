#  Nexvora - Innovative Digital Solutions

A modern, responsive business website for Nexvora, featuring a sleek dark theme, interactive services section, contact form with email integration, and AI assistant support.

##  Features

-  **Modern Dark Theme** - Clean, professional dark design
-  **Fully Responsive** - Works perfectly on all devices
-  **Image Slideshow** - Dynamic hero section with auto-sliding images
-  **Service Cards** - Showcases Web Development, App Development, AI Chatbot, and Graphic Designing
-  **Contact Form** - Send messages directly via email
-  **AI Assistant** - Integrated AI chatbot for instant support
-  **WhatsApp Integration** - Quick contact via WhatsApp
-  **Smooth Scrolling** - Seamless navigation between sections

##  Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Netlify Functions (Email handling)
